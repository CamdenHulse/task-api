# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1.0] - 2026-03-31

### Added
- Real-time crypto price dashboard showing BTC, ETH, and SOL prices from CoinGecko
- Live sparkline charts that build up over time with each 10-second refresh
- 24h percentage change badges with green/red coloring
- Market cap, 24h volume, and 24h price range stats per coin
- Countdown ring timer showing seconds until next data refresh
- Live clock display
- Price flash effect when prices change between refreshes
- Skeleton loading states while data loads
- Responsive grid layout (mobile, tablet, desktop)
- Dark terminal-style UI with monospace font
